# Gestión de Estudiantes

Sistema web para gestión de estudiantes desarrollado en PHP

## Características
- Sistema de login con usuarios quemados
- CRUD completo de estudiantes
- Interfaz responsive

## Tecnologías
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## Instalación
1. Clonar repositorio
2. Configurar base de datos en config/database.php
3. Importar sql/database.sql
4. Ejecutar en servidor local
